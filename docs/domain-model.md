## User
Простой пользователь системы.
Может создавать, редактировать и удалять СВОИ заметки.

## Admin
Вместо заметок контроллирует пользователей.
Заметок на экране нет вовсе, вместо них - простые пользователи.
Может редактировать и удалять пользователей, а также делать их админами.

## Note
Заметка с текстовым содержимым.
Может быть:
- в избранном
- архивной
- привязана к категории (в далеких планах)

## Category (в далеких планах)
Категория для группировки заметок.
Принадлежит пользователю.
У каждого свои группы.

## Краткое содержание функционала
Экран регистрации, пользователь регистрируется, входит в созданный аккаунт, заметок нет, создает заметку, может отредактировать ее
или удалить. Поиск заметок по названию. У заметок есть статус: активна/в архиве, архивная зачеркнута. Помимо поиска есть фильтр по статусу.
Заметку можно выполнить, а можно удалить, это разные вещи
